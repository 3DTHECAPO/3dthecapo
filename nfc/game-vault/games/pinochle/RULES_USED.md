# Rules Used - Pinochle

- Variant: 4-player partnership double-deck pinochle.
- Partners sit opposite each other. The 80-card deck is two standard 48-card pinochle decks with all 9s removed, leaving only A, 10, K, Q, and J.
- Deal: 20 cards to each player, no stock.
- First team to 500 wins. Minimum bid is 50. Highest bidder names trump and must hold a trump marriage (K + Q of trump).
- Meld occurs before trick-taking.
- Trick rank is A, 10, K, Q, J. Players must follow suit if possible, must beat if able, and must trump if void.
- Trick counters are A, 10, and K at 1 point each. Last trick is worth 2 points, so total trick points equal 50.
- If the bidding team fails to make its bid, it is set and loses the bid amount.
- If both teams pass 500, the higher score wins. Hands continue until a team reaches 500.
- Excluded from this variant: 9s, stock, and alternate scoring models.
