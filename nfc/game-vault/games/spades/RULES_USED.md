# Spades Rules Used

- Source consulted: Bicycle Cards Spades rules and standard partnership Spades scoring references.
- Variant implemented: four-seat partnership Spades with North/South versus East/West.
- Fully implemented: 52-card deck, 13-card deal to each seat, bidding, contracts, trick taking, follow-suit enforcement, spades-broken rule, team scoring, overtrick bags, 10-bag penalty, nil bid scoring, CPU fill in solo mode.
- Intentionally guarded/future: full four-human pass-and-play controls, Blind Nil, detailed multiplayer turn authority, and persistent live room state. Fan Challenge keeps the guarded room UI and does not claim full live sync.
