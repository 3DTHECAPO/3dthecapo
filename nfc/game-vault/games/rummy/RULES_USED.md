# Rummy Rules Used

- Source consulted: Britannica overview of rummy-family rules and standard basic two-player rummy references.
- Variant implemented: basic two-player draw-and-discard rummy foundation.
- Fully implemented: 52-card deck, 10-card hands, draw from stock or discard, meld validation for sets and same-suit runs, discard to end turn, CPU draw/meld/discard turn, visible CPU card backs, round completion, local point awards.
- Intentionally guarded/future: layoff rules, jokers, gin-specific knocking, multi-round match scoring, and live Fan Challenge synchronization. Fan rooms keep the guarded room UI without claiming full live sync.
