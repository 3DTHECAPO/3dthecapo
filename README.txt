# 3D THE CAPO — v21 (Root Drag & Drop)

This folder is **GitHub Pages friendly** and **root upload friendly**:
- Put everything in your repo root (same level as `index.html`)
- Or upload the contents to any web host

## Quick edit (drag & drop)
Replace images by dropping in a new file using the **same filename**:
- `merch_diamond_hoodie.jpg`
- `merch_diamond_tshirt.jpg`
- `merch_diamond_trucker.jpg`
- `merch_chip_hoodie.jpg`
- `merch_chip_tshirt.jpg`
- `merch_chip_trucker.jpg`
- `merch_classic_hoodie.jpg`
- `merch_classic_tshirt.jpg`
- `merch_classic_trucker.jpg`
- `watermark.png`

## Prices + descriptions
Open `script.js` and edit the `PRODUCTS` array.

## Checkout
The Checkout button is a placeholder link:
- Edit `script.js` → `checkoutBtn.href` to your Stripe Payment Link / Shopify checkout URL.
